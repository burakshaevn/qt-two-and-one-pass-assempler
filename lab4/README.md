# Однопросмотровый ассемблер для абсолютных программ

Программа моделирует работу однопросмотрового ассемблера для абсолютных программ, переписанная с C#/WPF на C++/Qt.

## Требования

- Qt 6.x
- CMake 3.16 или выше
- C++17 совместимый компилятор (GCC, Clang, MSVC)

## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск

После сборки выполните:

```bash
./Assembler
```

или на Windows:

```bash
.\Assembler.exe
```

## Использование

1. Введите исходный код ассемблера в левое верхнее поле
2. Введите таблицу кодов операций в левое нижнее поле
3. Используйте кнопки:
   - **Шаг** - выполнить один шаг ассемблирования
   - **Полный проход** - выполнить полную ассембляцию
   - **Сброс** - сбросить состояние ассемблера

Результаты отображаются в правых полях:
- Двоичный код
- Таблица символических имен (ТСИ)
- Ошибки (если есть)

## Структура проекта

- `assembler/` - логика ассемблера
- `helpers/` - вспомогательные классы (Parser, Comparer)
- `MainWindow.*` - главное окно приложения
- `main.cpp` - точка входа
